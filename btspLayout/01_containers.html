.container는 각 브레이크포인트 별로 최대 가로사이즈가 정해져있다.
1400 ~ :: 1320
1200 ~ 1400 :: 1140
992 ~ 1200 :: 960
768 ~ 992 ::720
576 ~ 760 :: 540
~ 576 :: 100%
.container-fluid는 각 브레이크포인트와 상관없이 항상 가로사이즈가 100% 이다.
100% < 576 (sm) < 540 : 720 : 960 : 1140 : 1320
100% < 768 (md) < 720 : 960 : 1140 : 1320
100% < 992 (lg) < 960 : 1140 : 1320
100% < 1200 (xl) < 1140 : 1320
100% < 1400 (xxl) < 1320
